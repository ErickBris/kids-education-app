What is this?

his app is for toddlers or Kids 1-3. Which provide education with sound and image.
 
This app Contains 4 Different Categories:
Alphabet from A – Z with Graphic and Sounds
Number 1-20 with Graphic and Sounds
Color with Graphic and Sounds
Shapes with Graphic and Sounds
 
FEATURES:

4 categories with Images and Sounds

Beautiful UI

Next Previous Buttons

Play Button

Flip pages by Gesture

Easy to Reskin

Document Included

This App comes with AdMob ready banner and interstitial Ads

 

HOW TO CHANGE ADMOB ID:

Go to Layout/res/values/Strings.xml

 
and change your banner ID
 
For Changing Interstitial ID:
 
Go to MainActivity.java files and change interstitial ID
 
name="adUnitId_intersitial">ca-app-pub-2691806550199991/3736815064</string>
 
 
 
HOW TO CHANGE PACKAGE NAME:
 
Right Click on Project and select Android Tools >> Rename Application Package and change your application package
 
 
 
 
HOW TO CHANGE APP NAME:
 

Go to Layout/res/values/Strings.xml and change your app name
